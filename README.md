# Game-of-Thrones-Survival-Prediction
This project aims to predict whether a character from the iconic TV series Game of Thrones will survive or die, based on various character attributes. Using machine learning techniques, the model analyzes historical data from the series to identify key factors that influence a character's fate in the ruthless world of Westeros.
## Project Goal
To build a binary classification model that predicts whether a character from Game of Thrones (A Song of Ice and Fire) survives or dies based on demographic, familial, and narrative features.
## Repository contents
* Data Analysis: Exploration and visualization of features, identifying key factors associated with survival.
* Data Preprocessing: Data cleaning, handling missing values, encoding categories (one-hot, label encoding), and normalization to prepare for modeling.
* Modeling: Application of various machine learning algorithms and comparison of their performance.
* Model Evaluation: Selecting accuracy metrics and comparing model results to choose the best approach.
## Technologies Used
* Language: Python
* Libraries: Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn
## Results
The model successfully identifies key survival factors including nobility status, house affiliation, number of book appearances, and family connections as the strongest predictors of character survival in the series.
